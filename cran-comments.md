## Package motivation

No CRAN package provides a unified interface for algorithmic fairness
assessment of clinical prediction models. clinicalfair fills this gap with
standard metrics (demographic parity, equalized odds, calibration
parity), visualization, threshold optimization, and intersectional
analysis. Designed for FDA AI/ML regulatory compliance.
